FAQ

Common questions

Is it free?
Yes, completely free. No subscription, no premium tier, no hidden costs. Ever.
Do I need an API key?
No. The AI categorization runs on a shared backend at no cost to you. Just install the extension and click Organize.
Will it delete my chats or projects?
No chats are ever deleted. Projects that are left completely empty after organizing are removed to keep your sidebar clean, but every conversation stays intact in your chat history.
Does it work in languages other than English?
Yes. The keyword wordlist covers Dutch, French, German, Spanish, Italian, Polish, Russian, Arabic, Hindi and more. The AI fallback works in any language.
Do I need to be logged in to Claude.ai?
Yes. The extension uses your active Claude.ai browser session to read and move chats. Open claude.ai, log in, then click the extension icon.
Can I undo an organize run?
Not automatically, but you can move chats back manually in Claude.ai. That's why the preview step exists: review the plan carefully before clicking Apply.
How do I organize my Claude chats?
Install the Chat Organizer extension, open Claude.ai, and click Organize. It scans every unorganized chat, picks a project for each one using keyword matching and AI, and moves them all in one click once you approve the plan. No manual sorting, no API key.
Is there a tool to manage and organize hundreds of Claude conversations?
Yes. Chat Organizer is a free Chrome extension made to manage and organize large Claude.ai chat histories. It sorts hundreds of conversations into projects automatically, so you can find an old chat in seconds instead of scrolling forever.
Does it work with ChatGPT?
Yes. Chat Organizer works on both Claude and ChatGPT. Pick the platform in the popup (it auto-detects the tab you are on) and organize either sidebar the same way, with the same one-click preview. See the ChatGPT guide for details.
